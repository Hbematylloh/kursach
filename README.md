## О проекте

Веб-приложение для учёта аудиторного фонда учебного заведения. Позволяет управлять аудиториями, их оборудованием, расписанием занятий, а также искать свободные аудитории по различным критериям.

### Возможности

- **Управление аудиториями**: добавление, редактирование, удаление
- **Учёт оборудования**: проекторы, компьютеры, кондиционеры и т.д.
- **Расписание занятий**: создание и просмотр, защита от пересечений
- **Поиск свободных аудиторий**: по дате, времени, вместимости, оборудованию
- **Отчёты**: выгрузка данных в CSV формате
- **Статистика**: общая информация о загруженности

### Требования

- Python 3.8 или выше
- PostgreSQL 15+ (опционально, можно использовать SQLite)
- Git

### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/Hbematylloh/kursach.git
   cd classroom-system
    ```
2. **Создайте виртуальное окружение**

    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate
    ```

3. **Установите зависимости**

    ```bash
    pip install -r requirements.txt
    ```

4. **Настройте базу данных**

    Установите PostgreSQL

    Создайте базу данных classroom_db через pgAdmin

    Настройте подключение в config.py

5. **Запустите приложение**

    ```bash
    python app.py
    ```
